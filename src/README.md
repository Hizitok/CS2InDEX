# Project Overview

该项目实现了一个基于智能合约的去中心化永续合约交易系统，主要通过 Factory、Vault、Pool、IndexOracle、Liquidation、PositionNFT 和 Router 等合约协作完成资金管理、订单簿撮合、资金费率结算、清算操作以及用户交互。系统通过使用 NFT 来管理用户的仓位，以增强系统的透明度和便捷性。

---

## Contract Structure

- **Factory.sol**
  工厂合约。 用于创建相关合约，并创建相关池子

- **Vault.sol**  
  资金托管合约。该合约负责管理用户的保证金和资产。用户的资金会通过该合约进行托管，确保资金的安全性和合约之间的正确交互。

- **Pool.sol**  
  池子订单簿（OrderBook）核心合约。该合约负责存储和管理所有的买卖订单。订单簿使用平衡树数据结构来优化查询和订单匹配的效率。

- **Liquidation.sol**
  清算引擎合约。该合约负责监控仓位保证金率，当剩余保证金低于20%时触发强制平仓。采用两步清算模型：先判定触发条件，再以破产价格在订单簿中挂限价单，由正常撮合流程完成清算。

- **IndexOracle.sol**
  价格预言机合约。 用于接收链下数据，计算某阶段资金费率，并执行资金费率操作

- **Router.sol**  
  用户交互合约。用户通过该合约进行下单、平仓、查询仓位状态等操作。Router合约实现了与用户的交互逻辑，确保用户的操作能够被正确地发送到核心合约进行处理。

- **PositionNFT.sol**  
  仓位信息管理 核心合约。该合约负责存储和管理所有的仓位数据，在ERC721标准下仅可由Pool合约进行创建和信息更新。

## 使用NFT管理仓位

该系统使用NFT作为每个用户仓位的代币表示。每个仓位对应一个唯一的NFT，确保仓位管理的透明性和去中心化。具体操作流程如下：

### 1. **开仓**

- 用户提交开仓请求，系统收取相应的保证金。
- 保证金直接由vault中的用户余额转出到pool合约，由vault托管
- 系统会铸造一个新的NFT，作为该仓位的唯一标识。
- 此NFT会被存储在用户账户中，且在**Pool**合约的订单簿中挂单，等待匹配。
- 一旦订单成交，NFT的状态会被更新，标记为已成交，并进行相应的仓位调整。

### 2. **平仓**

- 用户提交平仓请求，系统会更新对应的NFT状态。
- 平仓完成后，NFT会显示为已平仓状态，并从订单簿中移除。
- 用户的资金会根据平仓结果返还至Vault合约。
- 若有提现需求，资金由vault合约返还给用户

## 数据结构

- **OrderBook 平衡树** 存储于 **Pool合约**
  该平衡树（Order Statistics Tree）数据结构用于高效存储和管理订单数据。它能快速处理订单的插入、查询和匹配，使交易过程更加高效。

- **Liquidation 平衡树** 存储于 **Liquidation合约**
  清算引擎使用平衡树按触发价格排序所有开放仓位。当市场价格触及触发价格时，系统自动执行强制平仓，防止账户爆仓。

---

## 操作流程

1. **开仓操作**  
   用户提交订单，系统验证保证金是否足够。如果满足条件，会铸造一个NFT，并在订单簿上挂单。订单的状态在成交后会更新。

2. **平仓操作**  
   用户提交平仓请求，系统会根据当前市场价格计算平仓金额，并更新NFT的状态。成功平仓后，用户的资金会返还至Vault合约，NFT会移除或标记为已平仓。

3. **自动清算**
   在市场价格剧烈波动或资金费率持续累积时，当仓位保证金率低于20%，Liquidation引擎会自动触发强制平仓，以破产价格在订单簿中挂限价单完成清算。

---

## 合约间交互

- **Vault** 合约通过**Router**合约与用户交互，确保用户的资金在进行交易前被安全托管。
- **Pool** 合约的订单簿与**Liquidation**合约的清算机制配合工作，确保订单的匹配和资金的安全。
- 用户通过**Router**合约提交的交易请求会首先进入**Pool**合约的订单簿，等待匹配，匹配后订单状态更新，**Pool**合约通知**IndexOracle**更新VTWAP数据，并向**Liquidation**引擎注册新开仓位。

---

## 安全性

- **资金托管**：所有资金都会存储在**Vault.sol**合约中，合约逻辑确保资金的安全，防止用户资金被篡改或挪用。
- **清算机制**：通过**Liquidation.sol**合约的自动清算机制，系统能够在市场剧烈波动或资金费率累积时自动执行仓位清算，防止账户过度亏损。
- **透明度**：使用NFT来表示仓位，每个NFT代表一个唯一的仓位，确保交易和资金流向的透明性。
